## 🚀 Getting Started

### Quick Start
```bash
# Install Fastest
pip install fastest

# Run tests
fastest tests/

# With virtual environment
source .venv/bin/activate
fastest tests/
```

### Key Features
- ⚡ 88x faster test discovery than pytest
- 🏃 2.1x faster test execution
- 🔄 Smart caching system
- 🎯 Parallel execution support
- 🔧 pytest compatibility for common features
